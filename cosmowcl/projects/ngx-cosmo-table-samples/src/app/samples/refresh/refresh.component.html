Basic table with different refresh policies.

Allowed policies are:

  • NO AUTO REFRESH
  • INTERVAL REFRESH
  • ON-PUSH REFRESH
  • REFRESH PROGRAMMATICALLY
  • OBSERVABLE AS DATA INPUT

This table has refreshStrategy TIMED and refreshes every 2 seconds. Watch the 'Last updated' column.

This table has refreshStrategy ON_PUSH and refreshes when the refreshEmitter emits a value. Watch the 'Last updated' column.

This table has refreshStrategy NONE and does not refresh automatically. Press the button to refresh programmatically. Watch the 'Last updated' column.

This table has an observable as data input. It refreshes automatically every time the observable input emits. Watch the 'Last updated' column.
