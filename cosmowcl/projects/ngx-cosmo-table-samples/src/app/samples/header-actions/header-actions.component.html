Basic table with action buttons.

Actions are conditionally enabled according to the property 'activationCondition'.

Button 'Nothing' is always enabled (activationCondition: CosmoTableActionActivationCondition.ALWAYS)

Button 'Alert' is enabled if a one or more elements are selected (activationCondition: CosmoTableActionActivationCondition.MULTIPLE_SELECTION)

Button 'New' is enabled if no elements are selected (activationCondition: CosmoTableActionActivationCondition.NO_SELECTION)

Button 'Clone' is enabled if a single element is selected (activationCondition: CosmoTableActionActivationCondition.SINGLE_SELECTION)

Button 'Delete' demonstrates arbitrary conditions: it is enabled if all selected elements are NOT ACTIVE (status column) (activationCondition: CosmoTableActionActivationCondition.DYNAMIC + actionStatusProvider)
